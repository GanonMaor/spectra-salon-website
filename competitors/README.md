# Competitor Research Hub

This folder contains structured competitor research for:

- Salon scheduling and booking software competitors
- Adjacent companies relevant to Spectra's smart weighing system

Each competitor should have its own subfolder with a dedicated `README.md`.

## Folder Structure

```text
competitors/
  README.md
  feature-comparison.md
  pricing-comparison.md
  glossgenius/
    README.md
  <competitor-name>/
    README.md
```

## Research Standards

For every competitor profile, include:

1. Company overview
2. Product focus
3. Feature breakdown by category
4. Strengths and strategic positioning
5. Potential gaps/opportunities for Spectra
6. Source links

## Naming Convention

- Use lowercase folder names with hyphens: `glossgenius`, `booksy`, `fresha`, `square-appointments`
- Keep all research in English

## Next Step

After adding multiple competitor profiles, update `feature-comparison.md` to build a cross-competitor matrix and define where Spectra should outperform the market.

Use `pricing-comparison.md` to track subscription models, processing fees, and add-on economics across competitors.
